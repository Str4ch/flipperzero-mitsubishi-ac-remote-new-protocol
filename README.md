## Building
```shell
# Clone repository
git clone https://github.com/Str4ch/flipperzero-mitsubishi-ac-remote-new-protocol.git
cd mitsubishi_ac_remote

# Build the application
ufbt build

# Build and launch the application
ufbt launch
```
